Minhas feramentas

HTML

É a linguagem fundamental para a criação e estruturação de páginas web. Utiliza marcações para definir o conteúdo de uma página, como textos, imagens, links e outros elementos, organizando-os em tags que indicam ao navegador como exibir o conteúdo.

CSS

Enquanto o HTML define a estrutura de uma página web, o CSS é usado para estilizar e apresentar esse conteúdo. Com CSS, é possível definir cores, fontes, layouts, espaçamentos e outros aspectos visuais, separando assim a estrutura (HTML) da apresentação (CSS) da página.

JAVASCRIPT

É uma linguagem de programação utilizada principalmente para criar interatividade em páginas web. Ela permite que você manipule o conteúdo HTML e CSS, adicionando comportamentos dinâmicos como validação de formulários, animações, atualizações de conteúdo sem recarregar a página, entre outras funcionalidades.

JAVA

Ao contrário das outras três linguagens mencionadas, Java não está diretamente relacionada ao desenvolvimento web front-end, mas é uma linguagem de programação versátil e amplamente utilizada. Java é conhecida por sua portabilidade e é usada principalmente no desenvolvimento de aplicativos de desktop, aplicativos móveis (Android), sistemas embarcados e servidores web, através de frameworks como Spring.

PYTHON

Python é uma linguagem de programação de alto nível e de propósito geral, ou seja, foi projetada para ser simples e eficiente em várias aplicações. Criado por Guido van Rossum e lançado em 1991, Python é uma das linguagens mais populares do mundo, tanto para iniciantes quanto para programadores experientes.

PHP

é uma linguagem de programação amplamente utilizada para desenvolvimento web. Ela é executada no lado do servidor, o que significa que o código PHP é processado no servidor e o resultado é enviado ao navegador do usuário em forma de HTML.
